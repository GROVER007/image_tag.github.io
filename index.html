Image and Font Style

img tag

HTML < img > tag is used to add image inside webpage/website. Nowadays website does not directly add images to a web page, as the images are linked to web pages by using the < img > tag which holds space for the image
< img src="images/image1.svg" >
big tag

The < big > tag in HTML is used to increase the selected text size by one larger than the surrounding text. In HTML 5, it can be used by CSS
< p > This is < big > big < /big > tag < /p >

This is a big tag
small tag

The HTML < small > tag makes text one font size smaller in the HTML document. This tag is also commonly referred to as the < small > element
< p > This is < small > small < /small > tag < /p >

This is a small tag
mark tag

The HTML < mark > tag is used to mark or highlight text that is of special interest or relevance in an HTML document. Browsers traditionally render the text found within the < mark > tag as text with a yellow background color. This tag is also commonly referred to as the < mark > element
< p > I am < mark > Lovish Grover < /mark > < /p >
